📖 Guía del Sistema

CarWash Pro — Cómo funciona cada sección

⚡ Dashboard
Panel principal con resumen de todo el negocio. Muestra ingresos, ganancias, número de servicios y ventas de tienda.

Selector de período: cambia entre Día, Semana, Mes, Semestre o Año. Usa el campo de fecha para ir a períodos anteriores.

Las gráficas se actualizan automáticamente al cambiar el período.
🚿 Registrar Servicio
Aquí registras cada lavado que realizas en el día. Selecciona el tipo de servicio y el tipo de vehículo.

El sistema calcula el precio sugerido automáticamente (precio base × multiplicador del vehículo). Puedes ajustarlo si cobras diferente.

Al guardar, el sistema descuenta los insumos del inventario y registra el costo real de los productos usados.
📋 Tipos de Servicio
Define los servicios que ofreces: Express, Completo, Motor, Vestiduras, etc.

Cada servicio tiene un precio base. El precio final depende también del tipo de vehículo.
🏎️ Tipos de Vehículo
Define los tipos de vehículo con su multiplicador de precio.

Ejemplo: si el lavado Express cuesta $80 y la camioneta tiene multiplicador 1.5, el precio sería $80 × 1.5 = $120.
🔗 Insumos por Servicio
Aquí defines cuánto de cada producto se usa en cada tipo de lavado.

Ejemplo: el Lavado Express usa 0.1 litros de shampoo. Con 10 litros en stock puedes hacer 100 lavados.

Esta configuración es clave para que el sistema calcule automáticamente el costo de insumos por servicio.
🧴 Insumos (Inventario)
Lista de todos los productos que usas para lavar autos: shampoos, ceras, desengrasantes, etc.

Puedes ver el stock actual y cuántos servicios puedes hacer con lo que tienes.

Si el stock baja del mínimo que definas, aparece una alerta en el Dashboard.
🛒 Compras / Entradas de Insumos
Registra cada vez que compras insumos de trabajo (shampoo, ceras, etc.).

Al guardar, el stock del insumo se actualiza automáticamente sumando la cantidad comprada.
🏭 Proveedores
Directorio de tus proveedores de insumos y mercancía. Guarda nombre, teléfono, contacto y email.

Al registrar compras o entradas, puedes vincularlas a un proveedor para tener historial.
🛍️ Tienda — Productos en Venta
Inventario de mercancía que vendes por separado: aromatizantes, microfibras, accesorios, etc.

Cada producto tiene precio de compra y precio de venta. El sistema calcula el margen de ganancia automáticamente.

Puedes ver el valor total de tu inventario en la sección principal de tienda.
🏷️ Registrar Venta (Tienda)
Registra la venta de un producto de la tienda. El sistema pre-llena el precio de venta sugerido.

Al guardar, el stock se descuenta y se registra la ganancia exacta (precio venta − precio compra × cantidad).
📥 Entradas de Tienda
Registra cuando recibes mercancía nueva para la tienda. Al guardar, el stock del producto aumenta y se actualiza el precio de compra más reciente.
👤 Clientes y Vehículos
Directorio de clientes con su información de contacto y los vehículos registrados.

Por cada cliente puedes registrar:
  • Nombre y WhatsApp (con botón directo a chat)
  • Uno o varios vehículos con placa, marca, modelo, año, color y tipo
Al registrar un servicio, puedes seleccionar el cliente y su vehículo — la placa y el tipo se autocompletan automáticamente.

En el perfil del cliente puedes ver todo su historial de servicios y cuánto ha gastado en total.
